# MayoRunner

A fun and exciting endless runner game where you play as a jar of mayonnaise!

## Gameplay

- Dodge obstacles by swiping left and right.
- Jump over hurdles by swiping up.
- Collect coins to unlock new characters and power-ups.
- The game gets progressively harder the longer you survive.

## Features

- Endless running fun!
- Simple and intuitive controls.
- Unlockable characters and power-ups.
- High score leaderboard.
- Fun and quirky graphics.

## Installation

1. Clone the repository: `git clone https://github.com/FilipChalupa/MayoRunner.git`
2. Install the dependencies: `npm install`
3. Run the game: `npm start`
